# APS-web1 / Diosnei.Dev

Portfólio acadêmico desenvolvido em HTML, CSS e Bootstrap.

## Objetivo

Criar um portfólio pessoal para apresentar formação, currículo, projetos e contato, aplicando conceitos de estrutura HTML, estilização CSS, Bootstrap e navegação entre páginas.

## Tecnologias

- HTML
- CSS
- Bootstrap

## Páginas

- `index.html` - página inicial do portfólio.
- `curriculo.html` - currículo em formato web.
- `portifolio.html` - projetos e práticas acadêmicas.
- `contato.html` - canais de contato.

## O que o projeto pratica

- Estrutura de site multipágina.
- Navegação entre páginas.
- Layout responsivo com Bootstrap.
- Cards e seções de apresentação.
- Organização de conteúdo profissional.

## Como visualizar

Abra o arquivo `index.html` no navegador ou publique o projeto em uma hospedagem estática.
