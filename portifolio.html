Detalhes do Projeto

Como o portfólio APS foi planejado e desenvolvido.

Objetivo do Projeto

O objetivo deste projeto foi criar um portfólio pessoal para apresentar minhas habilidades, experiências e projetos de forma clara e profissional. O portfólio foi planejado para ser responsivo, acessível e fácil de navegar, utilizando as melhores práticas de desenvolvimento web.

Tecnologias Utilizadas

  • HTML5 semântico
  • CSS3 com Sass
  • Bootstrap 5 para layout e componentes
  • JavaScript para interatividade
  • Git para controle de versão

Aprendizados e Desafios

Responsividade

Aprendi a importância de começar o design pelo mobile para garantir uma experiência fluida em todas as telas.

Organização de Código

Descobri que separar HTML, CSS, JavaScript e imagens em pastas distintas facilita a manutenção e escalabilidade do projeto.

Acessibilidade

Entendi como usar atributos ARIA e práticas semânticas para tornar o site mais acessível a todos os usuários.
